# React Frameworks Assignment

This project demonstrates multiple React frameworks:

## Projects included
- Next.js app with dynamic routing and API fetching
- Remix app with nested routing
- Gatsby static portfolio site
- Expo mobile app with API data

## How to run each project

### Next.js
cd next-app
npm install
npm run dev

### Remix
cd remix-app
npm install
npm run dev

### Gatsby
cd react-frameworks-lab
npm install
npm run develop

### Expo
cd expo-app
npm install
npm start
